## What?
railsのgem『シンプルカレンダー』を使用した、日時予約表をつくってみた。
<br>
<br>

## Why?
ユーザーが予約できる日時を、あらかじめ確認できるように。
<br>
<br>

## 機能
- 1時間毎の予約の可否を○X表示
- かつ１週間分の表示
- ユーザー登録しなくても予約可能
- ユーザー登録すると、楽！
